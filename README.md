# Small Business IT Infrastructure & Security Audit

## Overview
This project documents a security and infrastructure audit performed for a small hospitality venue in Melbourne, Australia.

The objective was to improve network visibility, security posture, and operational documentation.

## Project Goals
- Identify all devices on the network
- Document infrastructure and management access
- Identify potential security risks
- Improve monitoring and operational visibility

## Technologies Involved
- Network infrastructure
- Wi-Fi networks
- POS systems
- Windows endpoints
- Device monitoring

## Key Work Performed
- Network device discovery
- IP address documentation
- Infrastructure diagram creation
- Security posture assessment
- Monitoring and alert implementation

## Outcomes
- Full infrastructure documentation created
- Improved credential management practices
- Device monitoring implemented
- Security risks identified and documented

## Skills Demonstrated
- Infrastructure auditing
- Network discovery
- Security assessment
- Technical documentation
- Monitoring implementation
